CLICK TO RUN — Content Engine (local image preview)
===================================================

Double-click any .bat file. It will generate PNGs into:
    content_engine\generated_posts\

No Telegram or API keys are required (real charts use Yahoo Finance).

Files:
  strategy.bat   — Real charts, diagrams, cheatsheets + one text card (sample strategies)
  news.bat       — News intel cards + headline samples
  tips.bat       — Quick tips cards (first 3 themes)
  diagrams.bat   — All 12 diagram templates
  all.bat        — Runs strategy + news + tips + diagrams (takes longer)

Requirements: Python 3.11+ on PATH (try "py" or "python" from a terminal).

Project root is detected automatically (two folders above this file).
